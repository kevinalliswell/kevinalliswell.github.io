## 为什么选择 Hugo？

Hugo 是一个用 Go 语言编写的静态网站生成器。选择它的原因：

1. **超快的构建速度** - 毫秒级构建
2. **简单的部署方式** - 可直接部署到 GitHub Pages
3. **丰富的主题** - 社区有大量优质主题
4. **强大的 Markdown 支持** - 写作体验极佳

## 博客功能特点

- 🚀 基于 Hugo + PaperMod 主题
- 📱 响应式设计，移动端友好
- 🌓 自动深色模式
- 🔍 内置搜索功能
- 📊 阅读时间估算
- 🔖 文章目录导航

## 文章分类

1. **工作笔记**
   - 技术学习总结
   - 项目开发日志
   - 问题解决方案

2. **热点新闻**
   - 技术趋势分析
   - 行业动态观察
   - AI 发展动态

3. **生活随笔**
   - 日常思考
   - 读书笔记
   - 生活记录
